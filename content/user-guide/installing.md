# Installing DitaForge

DitaForge can be installed on Windows, macOS, and Linux. Download the latest release from the official website and follow the platform-specific instructions below.

Navigate to our download page for downloads for all operating systems.

## Windows

1. Download the installer.
2. Run the installer and follow the prompts.

## MacOS

1. Download the `.dmg` file.
2. Drag DitaForge to your Applications folder.

## Linux

1. Download the `.AppImage` or use the package manager.
2. Make the file executable and run it.
